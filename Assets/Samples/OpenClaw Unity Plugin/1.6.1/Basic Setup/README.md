# Basic Setup Sample

This sample demonstrates the basic setup of OpenClaw Unity Bridge.

## Contents

- `OpenClawDemo.unity` - Sample scene with pre-configured bridge

## Usage

1. Import this sample via Package Manager
2. Open `OpenClawDemo.unity`
3. Configure your gateway URL in `Window > OpenClaw Plugin`
4. Enter Play Mode
5. Chat with OpenClaw!

## Scene Contents

- **OpenClaw Plugin** - Pre-configured bridge component
- **Main Camera** - Standard camera
- **Directional Light** - Scene lighting
- **Sample Objects** - Some primitives for testing
